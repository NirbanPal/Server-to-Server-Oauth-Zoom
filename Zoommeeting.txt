To create zoom meetings there is dictionary called payload in the createMeeting method. we have to pass the parameters like below. For instings here if i want to modify the timezone to Asia/Kolkata then the payload will be like this 

        payload = {
            "topic": topic,
            "duration": duration,
            'start_time': f'{start_date}T10:{start_time}',
            "type": 2,
            "password":password,
            'settings':{
                'join_before_host': True,
                'timezone': 'Asia/Calcutta' 
            }
        }
the time zone is in the settings cause you can see below(in the below json structure) the timezone is under a dictionary whose key is settings.
